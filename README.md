# World Cup 2026 — Live Forecast

A self-updating, calibrated probabilistic forecast for the FIFA World Cup 2026.

**Live dashboard → https://ilan-07.github.io/wc2026-live/**

A structural match-simulation model — hierarchical Bayesian / Dixon-Coles ratings plus a
Monte-Carlo tournament simulation — blended with the betting market, conditioned on results as
they happen and re-fit automatically when new data arrives. Validated with leave-one-tournament-out
backtesting across 9 tournaments and calibration checks (SBC, ECE).

The page refreshes itself; numbers update through the tournament.

_Informational only — not betting advice._
